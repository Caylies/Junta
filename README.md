# Junta

Junta is an upcoming FNF engine written in Luau for Love2D.

## Compiling Junta

- Install `rokit`.
- Run `rokit install`.
- Run `kaledis compile`

## Hosting Documentation

- Install `uv`.
- Run `uv run mkdocs serve`.

## Running Junta Editor

- Install `uv`.
- Run `uv run app`.

## Styling

### Luau

- Install `rokit`.
- Use `rokit install` to install StyLua and Selene.
- Run `stylua src`.
- Run `selene src`.

### Python

- Install `uv`.
- Run `uv sync`.
- Run `ruff check app`.
- Run `ruff format app`.
